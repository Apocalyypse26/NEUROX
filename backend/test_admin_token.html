🔐 NEUROX ADMIN TOKEN EXTRACTOR

Follow these steps to get your JWT token for admin testing:
1. Login to NEUROX - Make sure you're logged in with your admin account
2. Open Developer Tools - Press F12 or Right-click → Inspect → Console tab
3. Click Button Below - Click "Extract Token" to get your JWT
⚠️ IMPORTANT: Only use this token for testing purposes.
Never share your JWT token with anyone!
4. Test Admin Access - Copy the token and use it in test_admin_verify.py
